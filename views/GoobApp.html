Goob Music App Design


Time Taken: 3.5hr | Tools Used: Figma


Link to Figma Workspace

This project showcases Goob, a music app design I created using Figma to demonstrate my UI/UX design skills. The app features a sleek black base color, with red and green accents to highlight key interactive elements and actions.


Through this project, I learned how to use Figma to create visually appealing and functional designs. It also taught me how to apply color effectively to emphasize important elements and enhance the user experience.


This project taught me how to use color to emphasize what needs emphasis and the importance of layout to ensure readability.
