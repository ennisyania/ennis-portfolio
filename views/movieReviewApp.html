Description

This is a basic Movie Reviews app that I created where users are able to write reviews and add new movies into a database.




The video above demonstrates how the app can store reviews from different users for each individual movie in the database. The app also allows users to add, edit, and remove movies from the database. Additionally, reviews can be edited and deleted as needed.


This was made using EJS


This project taught me the basics of how websites interact with databases, allowing for live editing within the website. I also learned the importance of color schemes, as they affect readability and influence the overall feel of the website, enhancing the user experience.
