Travel List Website

This is a travel list website that help keeps track of items to pack for a trip.

Time Taken: 30min | Tools Used: ReactJS


This video showcases a simple travel list app designed to help users keep track of the items they need to pack. The app allows users to sort the list by item name, quantity, and packing status. At the bottom of the page, users can see how much of the list is completed, providing a clear overview of their progress.


Developed using ReactJS, this project, though simple, gave me the opportunity to refine my list sorting skills within the context of web development. Additionally, I learned how to incorporate mathematical calculations into the code to automatically track and display the user's packing progress, adding convenience and functionality to the app.
