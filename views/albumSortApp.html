Album Search Mobile App

This is a mobile sorting app that sorts by different variables using user input.

Time Taken: 2hr | Tools Used: ReactJS


The video above showcases a simple search app that organizes album releases based on title, artist, or genre. The app ensures that user input matches regardless of capitalization, making it more intuitive and user-friendly. I also designed the app's aesthetics using CSS, giving it a more polished and visually appealing look compared to basic HTML styling.


This project was developed using ReactJS, a powerful library for building websites and apps. To ensure a seamless mobile experience, I used a virtual mobile device during development to test how the app would look and function on smaller screens, guaranteeing usability. Additionally, this was my first time connecting a mobile app to an existing online database, which was a rewarding learning experience and a valuable skill to acquire.


Through this project, I learned the importance of optimizing user experience on mobile devices and gained a deeper understanding of database integration. It also improved my problem-solving skills as I debugged and ensured the app's functionality matched user expectations. This experience has strengthened my confidence in using ReactJS and laid the foundation for future projects involving dynamic and responsive web applications.
