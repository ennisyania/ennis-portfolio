Pokemon Card Database Mobile App

This is a mutable mobile database app that stores Pokemon cards with their image and information.

Time Taken: 2hr | Tools Used: ReactJS


This video demonstrates a simple database app that stores Pokémon cards by their type. Each card includes an image and a name, and is categorized according to its type. The app allows users to add, edit, and delete cards in the database, offering flexibility in managing the cards.


This project was developed using ReactJS and built as a mobile app. I also used CSS to enhance the visual appearance and improve the overall user experience. The images used in the app were sourced from publicly available Pokémon card images found online.


Through this project, I gained valuable experience in integrating images into ReactJS mobile applications. Additionally, I learned how to enhance database functionality, specifically the ability to add, edit, and delete records. At the time of building this app, I was still relatively new to app development, and this project helped me sharpen my skills in mobile development and database management.
